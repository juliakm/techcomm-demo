What are GitHub Pages?

GitHub Pages is a free tool for creating personal homepage and project sites. The pages are written in GitHub-flavored Markdown and can be easily styled. Many technical project websites are powered by GitHub Pages today.

To generate a page, you'll need to have a free GitHub account.

This is a great option if you want to create a quick resume websites or would like to put information about a topic online.

What are its strengths and weaknesses?

Strengths

  • Free
  • It’s a quick process to create a site. You can have unlimited project pages.
  • Sites come with nice designs.
  • You do not have to worry about hosting a website.
  • GitHub is designed for collaboration. You can start a website about a topic and then have someone else easily contribute to it.
  • Your changes are versioned and you can easily switch back to a previous version of the site if you have an issue.
  • Site content written in Markdown, although you can also use HTML.
  • Most designs are mobile-friendly (responsive).
  • GitHub Pages come with Wiki

Weaknesses

  • You will need to know some HTML and Markdown.

  • The initial site is generated from Markdown into HTML. Future edits need to be made in HTML.

    • If you fall in love with Markdown, then there is a more robust option that will let you edit pages in Markdown in the future.

  • The process of saving your work can be confusing if you are not familiar with git.

    • Git is a version control system that programmers use to save their work and collaborate together.
  • Similarly, making changes to the site requires some familiarly with git.
  • You can’t easily change the theme.

What are some alternatives?

  • While there are a number of ways to create a website, let’s focus on hosted projects.
  • Wordpress.com lets you create websites easily.
  • Squarespace lets you create websites for a fee.

Quick demo

  • Create a GitHub account if you do not already have one.
  • Create a new repository.
  • Go to Settings > Launch automatic pages generator.
  • Enter your content.
  • Pick a theme.
  • Edit your existing page.
  • Re-edit your existing page.

Why should I use this again?

  • Free
  • Create pretty pages quickly
  • Collaborate easily
  • Simple versioning to roll back when you make changes
